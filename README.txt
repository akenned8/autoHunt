This program is for educational use only, and should never be used to skip learning the information in the hunter safety course.

Simple python script to step through modules of hunter safety course automatically. 
Alex Kennedy

Requirements: (see video link below for install instructions) 
Python3
Selenium 
Latest version of Google Chrome
Latest version of chromedriver 
An established account with the hunter ed website

Instructions:
watch this video to make sure your system is set up correctly to run selenium 
https://www.youtube.com/watch?v=Xjv1sY630Uc
open hunt.py and fill out 'YOUR USERNAME' and 'YOUR PASSWORD' accordingly. It should be good to go after this

Warning:
If Google Chrome updates, the chromedriver will be outdated. You'll have to delete chromedriver and install the latest one
